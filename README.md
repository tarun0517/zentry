# Zentry Clone

A pixel-perfect recreation of the award-winning Zentry website built with React, Vite, GSAP, and Tailwind CSS.

##  Live Demo
[View Live Demo](https://zentry-2026.vercel.app/)


## 🌟 About The Project

This project is inspired by the original Zentry website, known for its immersive storytelling, cinematic animations, and innovative web experience.

- Advanced GSAP animations
- ScrollTrigger effects
- Responsive layouts
- Interactive video experiences
- Modern React development practices
- Performance-focused frontend design

## ✨ Features

- Smooth GSAP animations
- Scroll-based transitions
- Video-driven storytelling
- Responsive design
- Interactive UI components
- Modern gaming-inspired aesthetic

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- GSAP
- React Icons

## 📦 Installation

```bash
git clone https://github.com/tarun0517/zentry.git
cd zentry
npm install
npm run dev
```

## 🔨 Production Build

```bash
npm run build
```
